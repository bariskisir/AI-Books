# The Meridian Cycle

The Meridian Cycle is the first AI Books series: a 10-book English-language science-fiction cycle centered on slow, expensive, politically constrained faster-than-light travel through meridian gates.

The series is designed as 10 standalone novels with shared continuity. Books 0-9 are complete with EPUB editions, closing the cycle through a faster convergence of the existing arcs rather than leaving the earlier long-series plan unresolved.

## Production Workflow

1. Maintain the shared universe bible and 10-book catalog.
2. Create a detailed outline before drafting each book.
3. Draft each novel chapter by chapter in TXT.
4. Update metadata, manifest, and progress after each chapter or major production step.
5. Self-edit after each book is complete.
6. Generate cover and EPUB immediately after each book passes validation.
7. Add completed books to the root README.
