# The God Game — Final Report

## Summary

The God Game is a complete religious thriller novel written for the AI-Books repository. It tells the story of Father Matteo Lombardi, a Jesuit scholar who discovers a lost gospel beneath the Vatican — a first-century manuscript written by Judas Iscariot that claims Jesus survived the crucifixion, married Mary Magdalene, and fathered a bloodline that continues to this day.

- **Metadata:** JSON metadata file with descriptions and paths
