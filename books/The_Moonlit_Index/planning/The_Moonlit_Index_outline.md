# The Moonlit Index — Outline

Elian Rowe archives moonlight collected by Lumen's memory towers. When one night vanishes, she and disgraced astronomer Sable Venn find evidence that the council has been removing painful civic memories. A child named Pip maps shadows that should not exist. The investigation leads to the original lunar engine and a choice between restoring every memory and allowing citizens to decide what truth they can carry together.
