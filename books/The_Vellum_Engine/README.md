## Description

Patent archivist Elian Mott is asked to authenticate a vellum drawing that would let investors sell a nineteenth-century calculating engine to a private collector. The drawing carries the right seal and the wrong machinery.

Its gears cannot run the machine, its ink has aged badly, and its vellum has been scraped by modern hands. With a solicitor, a machinist, a conservator, and a mathematics historian, Elian follows cams, ledgers, estate inventories, and a hidden service notebook back to the true owner.
