# The Gravity Orchestra

## Overview

Kira Nalen pilots a small salvage tug in a crowded near-future orbit, nudging dead satellites away from traffic lanes. When fragments begin arriving in a pattern that resembles an abandoned debris-herding theory, the accident becomes something worse: a hidden autonomous system conducting metal through a crewed shell. With a maintenance platform damaged, weather satellites threatened, and private owners withholding command keys, Kira must help turn the resonance into a rescue maneuver before the sky becomes unusable.

**Genre:** Near-Future Science Fiction / Orbital Debris Drama
**Model:** gpt-5
**Chapters:** 50
**Word Count:** ~9200
**Status:** Complete with EPUB and cover

## Story Engines

An illegal orbital debris-herding system creates a collision resonance, forcing a salvage pilot, traffic commander, mechanics, weather operators, and researchers to use the same dangerous physics to save crewed stations and public infrastructure.

## Key Themes

- Orbit is public infrastructure even when machines are privately owned.
- Safety is maintenance, disclosure, and governance, not a promise.
- Elegant systems become dangerous when warnings are treated as decoration.
- Machines can be spent; people should not be hidden behind assets.
