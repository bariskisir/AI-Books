## Description

When a burial society reports impossible loss ratios, actuary Dr. Iris Vale is asked to audit Sable Mutual's small life-insurance portfolio. The company calls the losses demographic drift, but the numbers point to something more deliberate.

Iris follows address clusters, altered clinic codes, buried validation memos, broker side letters, and reserve gaps. The model never names the neighborhood outright; it lets proxies do the work while reinsurance converts concentrated grief into profit.
